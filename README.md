# SchoolManagementSystem (IOS)

A modern IOS app for managing school-related tasks, built with SwiftUI. Designed for modularity, scalability, and maintainability in educational environments. Developed by Visor Team (2025).

## Features

- Jetpack Compose-based UI with Material Design 3.
- Modular architecture for easy feature expansion.
- Prepared for unit and instrumentation testing.
- Edge-to-edge UI support and modern Android components.
